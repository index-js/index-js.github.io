MySQL备份
2019-07-18 12:03:00 # 编程 # 服务器 # Docker # MySQL

导出备份

mysqldump -u{root} -p{password} {database_name} > backup.sql

备份恢复

mysql -h{host} -P{port} -u{root} -p{password} -C < backup.sql

在Docker容器外执行

docker exec -i {container_name} mysql/mysqldump …
